A diferencia de la fase de Ingeniería del Dominio, en esta fase no se utiliza EUnit para realizar las pruebas ya que la flexibilidad de dicha herramienta no se ajusta a las comprobaciones que deben realizarse. De tal forma se procede a crear todos los casos descritos en el cuadro \ref{app:table:pruebas} como modelos independientes y se prueba uno a uno si el código generado para los productos específicos se corresponde con el esperado.
