\section{Conclusiones}
%%% Qué hace exactamente mi proyecto
Este proyecto ha desarrollado una línea de productos software para hogares automáticos y/o inteligentes mediante la tecnología .NET en el lenguaje C\# a través de sus clases parciales. El principal objetivo que se busca cuando se realiza un tipo de proyecto como este, es el de reducir costes de desarrollo y mejorar la calidad de los productos software. No obstante, el presente proyecto no tenía como principal cometido desarrollar un línea de productos software que pudiese ser trasladada a un caso práctico real de inmediato. En cierto modo, se ha utilizado un caso de estudio como es el de la automatización de los hogares, debido a la gran variabilidad y elementos comunes que poseen, para encontrar las posibles ventajas y desventajas que se presentan cuando se trabaja con las clases parciales de C\# sobre la tecnología .NET para construir líneas de productos software.

Durante el desarrollo del proyecto se han encontrado problemas que se presentan cuando se utilizan las clases parciales como mecanismo principal para encapsular las distintas características de una línea de productos. Por ello se han desarrollado nuevos mecanismos que pueden ser utilizados como una alternativa para suplir estos problemas. Por lo que el principal objetivo del proyecto ha sido demostrar que utilizando los mecanismos desarrollados en el presente proyecto se pueden construir líneas de productos software utilizando .NET y las clases parciales de C\#.
