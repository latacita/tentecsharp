\section{Sumario}

Durante este capítulo se han descrito los conceptos necesarios para entender la motivación de este proyecto. Se ha descrito qué es una línea de productos software, las técnicas utilizadas para su desarrollo, las limitaciones de la orientación a objetos para su implementación y las ventajas de los lenguajes orientados a características. No obstante, tal como se ha comentado, en muchas situaciones o los lenguajes de programación orientados a características no están siempre lo suficientemente maduros como para ser transferidos a la industria, o las empresas no pueden permitirse el coste asociado a cambiar su lenguaje de programación habitual. Por tanto, una posible solución, sería intentar implementar líneas de productos software usando las construcciones proporcionadas por un lenguaje de programación de amplia difusión, tal como C\#. Por dicho motivo, en las siguientes secciones implementaremos una línea de productos software para hogares inteligentes usando las clases parciales de C\#. El siguiente capítulo describe la planificación general seguida para la realización de este proyecto.
